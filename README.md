# Word_Game
Here there are 5 fields. And the user can choose any 1 field. After that the user has to guss the word from that fields based on hints given letter by letter. 
User can guess letters from the word and the those letters can be filled on the blank spaces. And for wrong guess points are deducted. 
Similarily other 2 rounds are played and a cummalative score is presented with the remakrs to the user and 3 rounds are over.
Hemce, it takes 3 round to make it a complete 30 score.
